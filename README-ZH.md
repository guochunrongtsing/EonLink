# EonLink: 具身智能虚实融合系统 (动态版)

EonLink 是一个融合了自然语言推理、虚拟世界仿真和物理硬件控制的具身智能系统。它利用 LLM 作为“大脑”，配合 **10Hz 实时动态监控环**，实现了在变化环境下的鲁棒执行。

## 1. 技术架构 (动态升级)

系统采用响应式四层架构，旨在实现高保真推理和实时物理响应：

*   **用户交互层 (User Interaction Layer)**: 将自然语言意图（如“去把红色的杯子拿过来”）解析为高级目标。
*   **智能体核心 (Agent Core/大脑)**: 采用分层优先级链（NVIDIA NIM -> OpenRouter -> Gemini 2.0）进行跨模态任务分解与迭代自修正，确保系统的高可用性与强大推理性能。
*   **世界模型 (World Model/仿真器)**: 基于 React Three Fiber 构建的 3D 环境，不仅用于执行前的静态验证，还在执行过程中进行 10Hz 的实时物理一致性监控。
*   **动态硬件桥接 (ROS 2 / 10Hz 闭环)**: 高频控制环将验证通过的动作拆解为每秒 10 帧的微指令，实时响应环境和机器人位姿的变化。

## 2. 核心运行逻辑：感知-决策-行动 (Sense-Think-Act)

### 闭环动态调整
EonLink 从传统的“规划后执行”模式升级为“动态闭环”控制：
1.  **分解与仿真**: 初始目标被分解并进行快速静态验证。
2.  **实时感知 (10 FPS)**: 在执行阶段，系统以 100ms 为周期持续扫描环境，获取最新的机器人坐标和物体位置。
3.  **轨迹实时修正**: 机器人的移动不再是离散的点对点跳跃，而是平滑的插值轨迹。如果目标物体位置发生改变，控制环会自动更新航点参数。
4.  **反应式重新规划**: 如果在执行中检测到严重偏差（如目标被遮挡或路径受阻），系统会立即中断当前动作序列，发起新的重规划。

### 运动动力学
-   **10Hz 监控**: 每 0.1 秒同步一次虚拟状态与硬件反馈。
-   **动态避障站位**: 实时维护距目标 0.6 米的安全距离。
-   **机械臂姿态同步**: 抓取物体后，手臂会根据实时的载荷状态和动力学约束调整至“胸前持物”的最优姿态。

## 3. ROS 2 深度集成

EonLink 作为高级智能任务编排器，深度对接 ROS 2：
-   **Nav2 导航**: 实时发布和更新 `/goal_pose`，支持动态重定位。
-   **MoveIt 交互**: 基于伺服控制的关节同步，实现平滑的抓取与放置。
-   **感知流同步**: 通过 `/tf` 和感知话题（如 `/detected_objects`）驱动系统的 10Hz 实时观测。

## 4. 技术演进与核心规则

在系统开发过程中，我们通过以下核心逻辑加固实现了系统的进化：

- **近身操作强制约束 (Proximity Mandate)**: “大脑”现在受到严格的物理约束。除非先执行 `navigate_to` 动作到达物体的精确 (x,z) 坐标，否则任何操纵动作（拿起/放置/打开/关闭）均被视为无效。
- **状态感知规划 (State-Aware Planning)**: 系统能够避免“无效劳动”。如果环境状态显示机器人已经拿到了目标物体，大脑会自动跳过导航和拿起阶段，直接执行前往目的地的后续动作。
- **迭代自修正闭环 (Refinement Loop)**: 当仿真验证失败时（例如机器人试图远距离隔空取物），失败的反馈会被送回大模型。这形成了一个“会自省的大脑”，在尝试物理动作之前先从虚拟失败中学习并修正计划。
- **10Hz 动态插值**: 3D 世界模型中的运动在 10Hz 频率下进行同步。这确保了机器人的“心理状态”与“物理状态”始终保持同步，允许系统对环境的细微漂移做出反应。

## 5. 技术栈

-   **前端**: React 18, Tailwind CSS, Three.js (React Three Fiber).
-   **推理**: Google Gemini API, NVIDIA NIM (Llama-3).
-   **通信**: 基于 React Refs 和 Hooks 构建的 10Hz 内部高速控制环。

## 5. 启动与执行

1.  **安装依赖**: `npm install`
2.  **环境配置**: 设置 `GEMINI_API_KEY`。
3.  **启动服务**: `npm run dev`
4.  **操作观察**: 
    - 输入高级指令后，在日志中查看 **"Dynamic Control Loop (10Hz Monitor)"**。
    - 观察机器人以平滑轨迹在 3D 空间中移动，实时响应环境反馈。
